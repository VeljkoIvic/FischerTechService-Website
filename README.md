# Fischer Tech Service – Website

Diese Website präsentiert Fischer Tech Service als modernen und persönlichen 
Ansprechpartner für Servicearbeiten rund um Angelruten und Rollen. Im Mittelpunkt 
stehen Qualität, Präzision und ein direkter Draht zu den Kundinnen und Kunden.

## Über das Projekt

Fischer Tech Service bietet professionellen Service, Reparatur, Tuning und Beratung 
für Angelausrüstung. Die Website stellt die wichtigsten Leistungen übersichtlich vor, 
erklärt den Ablauf des Services und macht es einfach, über ein Kontaktformular eine 
Anfrage zu senden.

Besonders wichtig war bei der Umsetzung ein klarer und vertrauenswürdiger Auftritt. 
Die Inhalte verbinden handwerkliche Kompetenz mit einer einfachen, 
benutzerfreundlichen Struktur – professionell, schnell und auf allen Geräten nutzbar.

## Technische Basis

| Technologie | Verwendung |
|-------------|------------|
| [Astro](https://astro.build/) | Framework / Static Site Generator |
| [Bulma](https://bulma.io/) | CSS Framework |
| [Sass](https://sass-lang.com/) | Stylesheet-Erweiterung |

## Seiten

- **Startseite** – Überblick und Einstieg
- **Über uns** – Das Team und die Geschichte
- **Leistungen / Austausch** – Alle angebotenen Services
- **Galerie** – Einblicke in die Arbeit
- **Fragen** – Häufig gestellte Fragen
- **Impressum** – Rechtliche Angaben
- **Datenschutz** – Datenschutzerklärung

## Lokale Entwicklung

Abhängigkeiten installieren und Entwicklungsserver starten:

```bash
npm install
npm run dev
```

Produktions-Build erstellen:

```bash
npm run build
```
